# Documentation Images

This folder contains images used in the documentation.

## Required Images

| File | Description | Suggested Size |
|------|-------------|----------------|
| `mcgng-banner.png` | Main banner for README | 600x200 |

## Creating the Banner

The banner should include:
- Project name "MCG-NG" or "MechCommander Gold: Next Generation"
- A mech silhouette or similar BattleTech imagery
- Retro/nostalgic aesthetic matching 1998 era

## Notes

- Use PNG format for transparency support
- Keep file sizes reasonable (< 500KB)
- Ensure images are freely usable (original work or licensed)
